Some things change and some things stay the same!

Been a while since the last post back in.. err.. 2018.. and you'd think I'd have more things to add here. Amusingly, the only major new release the Badgers did since the last post, is technically an update to the same game as the last time - Snake-A-Roid - so I've had to add the date to the title to differentiate them here. I say update, but that's not being particularly generous as it was a fairly major rewrite and it clearly shows even between the screenshots of both pages. The 2016 version is still available ( along with the earlier mini ludumdare original and the build we took to 4TG ) so you can see the progression. Still, near enough a decade between releases is a little bit much, so lets try and get something else out a bit quicker!

Speaking of, I've been looking into ZX Spectrum development. Specifically using z88dk and writing it in C. It's been going well, and I've been thinking of bringing a few games over - first one being a port of Germies! - and then one that languished on the back burner, and one that didn't make it past design. The idea being to get them all done this year, but I've been poking at things since December, and it's June already and I haven't finished the first one! So we'll see how that goes.

GameMaker's latest LTS version finally got released, which was a mammoth amount of work. A _little_ bit late, but we got there in the end.

I've also been looking into JavaScript and TypeScript recently, so might actually dust the site down a little more! Though as always, finding spare time to do much of anything is never easy, so don't hold me to that!

And finally... in the intermediary years, various services changed their terms of services, shut down, etc... and the backups I had bitrotted.. so anything that wasn't actually archived up on GitHub has sadly been lost. Thankfully this was mostly just coursework binaries and my terrible mod files, but it is a bit of a letdown that the drives went, and the discs I had archived to either got lost or damaged, so much of the files that I hadn't yet uploaded here now never will be. I hadn't had time to go back and check things, so it's really only my own fault.

Again, I'll try and find time to clean the site up a bit; be that a redesign or just going through things and taking out bad links, etc..

Hopefully that won't take another seven years before I next update things!
